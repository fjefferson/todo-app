# ToDo App
A simple task application

### Using
- NodeJS
- MongoDB
- React

### Instalation
```
npm i
```

### Backend Dependencies (Service:3003)
- body-parser
- express
- mongoose
- node-restful
- nodemon
- pm2

### Backend Run
```
npm run dev
```
or
```
npm run prod
```

### Frontend Run
